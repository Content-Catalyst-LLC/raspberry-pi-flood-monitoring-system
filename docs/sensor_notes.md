# Sensor Notes

## Ultrasonic Water-Level Sensor

Useful for estimating the distance between the sensor and the water surface.

Measurement quality depends on:

- mounting geometry
- surface turbulence
- wind interference
- temperature effects

## Rain Gauge

Useful for estimating rainfall accumulation and event intensity.

## BME280

Useful for atmospheric context:

- temperature
- humidity
- pressure

## General Note

Flood-monitoring accuracy depends not only on code and hardware, but also on:

- calibration
- sensor placement
- weather protection
- power stability
- local threshold design
